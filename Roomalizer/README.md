# Roomalizer
### Instant “put it in a room” placement — early reflections + depth cues

**Roomalizer adds space cues fast**: early reflections, depth balance, and a controllable stereo room return.

It’s not a full reverb engine — it’s a placement tool that helps dry sounds sit in a mix.

---

## What it’s good at

- Making dry dialogue/vocals feel like they exist in a space
- Giving SFX a believable “room” without long tails
- Fast near/far placement decisions

---

## Controls

### 1) Placement
Near/far placement feel (reflection timing/energy).

### 2) Depth
Direct vs room depth balance.

### 3) Size
Room size (scales reflection timing).

### 4) Width
Stereo width of the room return.

### 5) Mix
Wet amount.

### 6) Tone
Brightness of the room return.

### 7) Mono‑safe
Reduces phase risk for mono playback.

---

## Quick Start

- Placement: ~50  
- Depth: 30–60  
- Size: 40–70  
- Width: 50–70  
- Mix: 20–50%  
- Tone: to taste  
- Mono‑safe: 30–70 if needed

---

## Safety / behavior notes

- Higher Width can be less mono-compatible. Use Mono‑safe if needed.
