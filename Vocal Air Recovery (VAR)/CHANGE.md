# Changelog

All notable changes to this repository are documented here.
This project follows an experimental, research-driven development model.

## v1.1.0
- Added detector floor control to Vocal Air Recovery to prevent air recovery
  from activating on room tone or stationary noise.
- Improved robustness when used after denoising.

## v1.0.0
- Vocal Air Recovery (VAR): perceptual HF restoration using
  level-invariant curvature-based detection.
