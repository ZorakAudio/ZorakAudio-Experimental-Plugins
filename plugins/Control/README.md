# Control

Control plugins generate, translate, or expose **motion, performance, and analysis-driven control data**.

## Current folders

- **GesturePad** — draw gestures and turn them into MIDI CC, note, or mixed-output performance data.
- **NeuroCV** — predictive audio-analysis CV field prototype with multichannel routing and read-only modulation lanes.

## Build note

`NeuroCV` currently ships in-tree as a documented prototype. Its folder contains `plugin.json.bak`, not an active leaf `plugin.json`, so treat it as a parked experiment rather than part of the normal build catalog until its metadata is restored.
