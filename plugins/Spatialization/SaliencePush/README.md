# SaliencePush

## What it is
SaliencePush is a **perceptual “sit back” tool**.

The goal is not simple broadband ducking. The goal is to reduce how much a source **grabs attention** while keeping its body and identity more intact than a blunt level pull would.

The current source mainly works by trimming upper presence / air and by controlling wide high-band energy. It can also follow an optional foreground reference on channels 3/4.

---

## Why use it
Use it when a source should stay present but less commanding.

Typical use cases:

- backgrounds that still feel too “look at me”
- vocals that should sit farther back
- SFX or foley that should support without poking through
- splashy wet layers that need to stop jumping forward

---

## Quick start
1. Choose the **Profile** that best matches the source.
2. Raise **Push** to move the source backward.
3. Use **Tame** to control clicky or pokey transients.
4. Use **Preserve** to protect gloss, stereo life, and naturalness.
5. Feed a foreground reference to 3/4 if you want the source to sit under another element instead of only self-governing.

---

## Main controls
### Profile
Hidden policy bias for source type. The current source offers Neutral, Vocals, SFX, Foley, and BG.

### Push
Main backwardness / sit-back control.

### Tame
How strongly sharp upper-band pokes are restrained.

### Preserve
How much naturalness, gloss, and stereo life are protected while pushing the source back.

### Output Trim
Final gain trim.

---

## Routing / notes
**1/2** = source to push back  
**3/4** = optional foreground reference / key

Without a reference, the plugin falls back to self-salience. With a reference, it behaves more like a perceptual margin keeper against the foreground.

---

## In one sentence
SaliencePush makes a source sit back by reducing attention capture, not by simply turning the whole thing down.
