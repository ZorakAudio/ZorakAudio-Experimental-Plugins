# Roomalizer

## What it is
Roomalizer is a **minimal early-reflection placer**.

It is not trying to be a giant reverb. The current source is built to suggest room size, depth, and placement with stable early reflections and a controlled stereo field, while deliberately avoiding an obvious long reverb tail.

---

## Why use it
Use it when a source feels too dry or too close, but you only need enough room information to place it—not a big washy verb.

---

## Quick start
1. Start with **Depth** to decide how far away the source should feel.
2. Set **Size** for the apparent room scale.
3. Adjust **Placement** and **Width** until the source sits naturally.
4. Blend with **Mix** and season with **Tone**.
5. Enable **Mono-safe** when low-end stability matters.

---

## Main controls
### Placement
Primary position / placement feel inside the room.

### Depth
The main near/far control.

### Size
Apparent room scale and ER spacing.

### Width
Stereo spread of the room field.

### Mix
Wet/dry blend.

### Tone
A simple voicing trim around the upper spectrum.

### Mono-safe
Keeps low-end image behavior safer by centering low-side energy.

---

## Notes
The current source includes a safety limiter so parameter moves and preset changes are less likely to spit out surprises.

---

## In one sentence
Roomalizer adds enough early-room information to place a sound without turning it into a full reverb effect.
