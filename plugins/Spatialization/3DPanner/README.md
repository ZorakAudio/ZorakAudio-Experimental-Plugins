# 3DPanner

## What it is
3DPanner is a **headphone-focused, mouse-first 3D panner**.

The current source combines several perceptual cues:

- microdelay / interaural timing
- head-shadow style darkening
- distance loss
- occlusion
- room and early-bounce cues
- a canvas-based UI for direct placement

It is not just a left/right pan with a “width” knob bolted on.

---

## Why use it
Use it when a sound should feel like it sits in a real headphone space instead of merely moving across the stereo field.

---

## Quick start
1. Insert 3DPanner on the sound you want to place.
2. Drag the orange source in the room canvas to set position.
3. Use the mouse wheel over the room to adjust **Distance**.
4. Use modifier-wheel shortcuts for fine control: Shift for Width, Ctrl/Cmd for Room, Alt/Option for Externalize.
5. Trim the result with **Output Trim**.

---

## Main controls
### Azimuth
Horizontal angle placement.

### Distance
How close or far the source feels.

### Width
How compact or spread the source image feels.

### Externalize
Pushes the sound farther out of the head.

### Room
Amount of room cue contribution.

### Room Size
Scale of the room-cue geometry.

### Occlusion
How blocked or screened the source feels.

### Micro Motion
Small movement for extra realism and life.

### Output Trim
Final gain trim.

---

## Notes
The current UI is part of the workflow. Dragging the source in the room is not a gimmick; it is the intended way to think about placement.

The cyan wall markers in the interface represent early bounce cues that help a sound feel externalized.

---

## In one sentence
3DPanner is for making a headphone source feel placed in a room, not just panned between speakers.
