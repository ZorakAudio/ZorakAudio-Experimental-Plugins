# Spectral

Spectral, tonal, EQ, stabilization, and texture tools.

## Plugins

- `BedRock` — BedRock (JSFX)
- `ERBTilt` — ERB Tilt (JSFX)
- `SpectralStabilizer` — Spectral Stabilizer (JSFX)
- `TSEQ` — Temporal Structural EQ (TSEQ) (JSFX)
- `Texture` — Texture (JSFX)

Each plugin leaf carries its own `plugin.json`, source entry, and optional tests/docs/assets.
