# Spectral

Spectral plugins here reshape tone, structure, body, texture, and perceptual balance in ways that are often more content-aware than plain EQ.

## Current plugins

- **Alias** — signal-derived body reinjection using intentionally aliased side paths
- **BedRock** — adaptive pitch-aware subharmonic engine
- **Contour** — applies a loaded texture against the live input contour
- **ERBTilt** — perceptual ERB tilt EQ with loudness compensation and roughness guarding
- **Sample** — load-and-go psycho multisampler
- **SpectralStabilizer** — SAFE excess-only tonal stabilizer
- **TSEQ** — time-scale detail shaper with per-bucket thresholds and knees
- **Texture** — performance texture engine driven by live input, sidechain, or MIDI
- **TextureXY** — XY-drawn texture instrument built around path playback

## Category note

This is the largest category because a lot of the catalog’s experiments live at the intersection of tone, texture, analysis, and psychoacoustics.
