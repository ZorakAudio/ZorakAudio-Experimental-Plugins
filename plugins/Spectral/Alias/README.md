# Alias

## What it is
Alias is a **signal-derived body reinjection tool** built from intentionally downsampled / aliased side paths.

The current source is not trying to hide the aliasing. It uses controlled aliased paths as a way to rebuild body, density, and low-mid support from the source itself. The comment block in the source describes it well:

> signal-derived body reinjection via intentionally downsampled / aliased side paths

It is especially relevant after pitch shifting or on sources that feel thin, hollow, or overly “clean.”

---

## Why use it
Use Alias when you want more:

- body
- density
- glue
- low-mid reinforcement
- “real stuff happening underneath the top”

---

## Quick start
1. Start with **Body Mix** and the three path weights: **48k**, **24k**, and **12k**.
2. Use **Seed HP** to decide how direct or detail-derived the reinjection should be.
3. Set **Body Focus** to place the emphasis region.
4. Use **Drive**, **Smart**, and **Density** to decide how assertive the reinjected path feels.
5. Use **Audit = Wet Only** when you want to hear just the derived body path.

---

## Main controls
### Body Mix and path weights
These decide how much of the aliased reinjection comes back and which decimation paths are doing the work.

### Seed HP
Lower settings allow more direct reinforcement. Higher settings make the reinjection derive more from upper/detail content.

### Body Focus
Centers the perceived body emphasis.

### Drive
Pushes the derived path harder.

### Smart
Balances how adaptively the current source manages the reinjection.

### Decohere
Adds micro-delay so the wet path does not comb too obviously against the dry path.

### Density
How dense and filled-in the derived body feels.

### Output Trim
Final overall level trim.

### Audit
Add to Dry or Wet Only, for checking what the body path is really doing.

### Inject Gain / Wet HP / Wet LP
Fine trims for the reinjected path itself.

## Notes
The current source comment notes that Alias is especially happy at **96 kHz host rate**, where the full 48k / 24k / 12k path idea makes the most sense.

## In one sentence
Alias turns controlled aliasing into a body-building side path instead of treating aliasing as a mistake.
